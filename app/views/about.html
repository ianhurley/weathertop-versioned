About

WeatherTop Inc. have created a modular weather station called the WeatherTop 1000. This lowcost device takes a set of meteorological readings and displays these on our Web Companion Application.

Version 3: RELEASE 1

This version launches and displays readings from a small number of weather stations.The latest weather reading provides more detailed information.
Each reading consists of:

  • Station Name
  • Weather Conditions - Code & Description
  • Temperatures in both Celsius & Fahrenheit
  • Wind Speed (kM/hr) & Beaufort Conversion
    • Wind Direction
    • Wind Compass
    • Wind Chill
  • Pressure (hPa) Number
This version also permits two additional features:
  • Add Station
  • Add Reading
